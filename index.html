UV-5R VFO Unlock

Unlock Your VFO Keypad

Download as .zip Download as .tar.gz View on GitHub

Purpose

To unlock those cheap Baofeng hand held radio keypads so you can enter out-of-band frequencies directly into the keypad on the go. Range is 130-180mhz and 400-520mhz. While I could open this range more the radio isn't capable of transmitting further than this.

Windows

Simply download and open the exe file under the bin folder. Be sure to run the correct version based on your firmware. Obtain your firmware with the following instructions.

Hold down the 3 key while powering on the radio

Linux

Ensure you have ruby installed as well as rubygems

gem install serialport

Just run the program, enter the serial port path and wait for patching to complete.
